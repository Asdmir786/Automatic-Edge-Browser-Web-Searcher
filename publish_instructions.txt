# How to Build a Standalone EXE for Automatic Edge Browser Web Searcher

1. Open PowerShell in your project directory.
2. Run one of the following commands depending on your target platform:

**For 64-bit Windows:**
```
dotnet publish ".\Automatic Edge Browser Web Searcher.csproj" -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true
```
- The 64-bit .exe will be in:
  `bin\Release\net9.0\win-x64\publish\`

**For 32-bit Windows:**
```
dotnet publish ".\Automatic Edge Browser Web Searcher.csproj" -c Release -r win-x86 --self-contained true /p:PublishSingleFile=true
```
- The 32-bit .exe will be in:
  `bin\Release\net9.0\win-x86\publish\`

## Notes
- No need to distribute queries.txt; it is built into the .exe.
- You can copy the .exe to any Windows machine and run it from cmd or by double-clicking.
- If you update queries.txt, rebuild the project to embed the new queries.
